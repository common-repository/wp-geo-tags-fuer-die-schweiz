WP Geo Tags fuer die Schweiz

== Description ==

Mit WP Geo Tags fuer die Schweiz koennen die Meta Tags geo.region, geo.placename, geo.position and ICBM fuer jeden einzelnen Artikel und jede einzelne Seite separat fuer die Schweiz angegeben werden. Support: [http://www.blumenversand.ag/](http://www.blumenversand.ag/)

== Installation ==

1. Extract `wp-geo-tags-fuer-die-schweiz.0.1.zip`.
2. Upload the `wp-geo-tags-fuer-die-schweiz`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
